OverviewThis document provides the detailed implementation specifications for applying the three core themes (Arcane, Neon, Cosmic) to AGI-CAD's 8 primary components.This file serves as the build guide for our AI dev team (Claude, GPT, Gemini).2. Component: Nexus GraphThe 6-layout symbolic graph visualization system.ThemeImplementation DetailsKey Effects1. Arcane BlueprintBackground: Animated, scrolling parchment texture (bg-[url(...)] animate-pan-slow).Nodes: Rendered as hand-drawn golden "ink" circles. Use a Three.js SpriteMaterial with a custom-drawn texture.Edges: Thin, golden lines (var(--color-glyph-line)), with a subtle "ink bleed" pulse animation on selection.Text: font-serif (e.g., 'EB Garamond'), text-text-header.text-shadow: 0 0 5px var(--color-glyph-glow-primary); on labels.Selected nodes have a "wet ink" Sprite bloom.2. Blade Runner NeonBackground: Dark grid background (bg-gray-900) with a subtle radial-gradient from below.Nodes: PointsMaterial (Three.js) with blending: AdditiveBlending. Color: var(--color-accent-primary) (e.g., bright cyan or magenta).Edges: LineBasicMaterial (Three.js) with transparent: true, opacity: 0.5.Text: font-mono (e.g., 'Fira Code'), text-accent-primary.filter: drop-shadow(0 0 8px var(--color-accent-primary)); on labels.Use UnrealBloomPass (post-processing) for intense glow on selected nodes.Holographic shader on info panels.3. Cosmic GrimoireBackground: Live Three.js starfield/nebula shader (see Shader-Library.md).Nodes: Three.js Mesh with SphereGeometry and ShaderMaterial. Nodes are "stars" or "planets" with custom shaders (e.g., a pulsing corona).Edges: LineDashedMaterial representing "constellations."Text: font-sans (e.g., 'Inter'), text-text-header.AdditiveBlending on node materials for a "lit-from-within" look.Heavy use of Bloom post-processing.Selected nodes have an animated ring mesh.3. Component: Learning Panel (Telemetry)The UI displaying FS-QMIX simulation data (fitness, latency, d_var charts).ThemeImplementation DetailsKey Effects1. Arcane BlueprintPanel: bg-bg-component (dark parchment) with border-2 border-amber-800 (looks like wood/gold inlay).Chart.js: borderColor: 'var(--color-accent-primary)' (gold). backgroundColor: 'var(--color-accent-primary-translucent)'.Labels: font-serif, text-text-body.box-shadow: inset 0 0 10px #00000050; on the panel to give it depth.Text has a subtle text-shadow.2. Blade Runner NeonPanel: Translucent glass effect. bg-bg-component-translucent (#0A192F90). backdrop-filter: blur(12px);. border border-cyan-500/50.Chart.js: borderColor: 'var(--color-accent-primary)' (cyan). pointBorderColor: '#fff'. pointRadius: 4.Labels: font-mono, text-accent-primary.box-shadow: 0 0 20px var(--color-accent-primary-translucent); on the panel.Chart lines have a filter: drop-shadow(...).3. Cosmic GrimoirePanel: bg-transparent. The panel "floats" over the cosmic background.Chart.js: borderColor: '#ffffff' (white/starlight). backgroundColor: 'rgba(255, 255, 255, 0.1)'.Labels: font-sans, text-white.text-shadow: 0 0 5px #ffffff80; on all text.The chart grid lines are rgba(255, 255, 255, 0.2) to look like constellation guides.4.
