Arcane Blueprint Theme: Core CSS VariablesThis section provides the semantic CSS variable definitions for the Arcane Blueprint theme. These colors are utilized by the NexusChat-NextJS-Component.jsx file via standard Tailwind classes (e.g., bg-accent-primary maps to var(--color-accent-primary)).1. Usage Examples in Nexus ChatThe "Activate Mass OAuth Sign-In" button uses bg-accent-primary and text-text-on-accent.The "Guidance from the Oracle" text is set to text-accent-primary and uses a border of border-accent-secondary/50 for a soft glow effect.The TuningSlider utilizes the gold color for its range input focus and value display.3. Font System--font-header: A stylized, serif font to convey a mystical, powerful feeling (e.g., a specific custom font included in your theme assets).--font-body: A modern, clean sans-serif font for readability and data display (e.g., 'Inter' or a similar legible typeface).
